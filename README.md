# Learn courses platform

## Description
Single page application for learning courses. The application is built with Vue 3(Composition API) and TypeScript. As a state manager I use Pinia. As UI library I use Tailwind.

## Project setup
```
npm install
npm run dev
```

## Tests
```
npm run test:unit
```
